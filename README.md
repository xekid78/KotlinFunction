# KotlinFunction
関数

## 処理
`sum_gokei()`関数を作り、その中で計算を行う。

## コード
```
fun sum_gokei() {
    println(5 * 8)
}

fun main(args: Array<String>) {
    sum_gokei()
}
```

## 出力結果  
```
40
```
